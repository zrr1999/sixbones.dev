有理分式分解的一种快捷方法

在学习信号与系统等学科时，我们经常会遇到有理分式分解的问题。 通常情况下，对于分母为若干一次式的乘积的情况，我们可以通过留数法快速分解， 然而，对于分母含有二次式的情况，教科书上却鲜有提及。 我在多次遇到次此类问题后，认为可能会存在更快速的方法，通过几天的研究，总结一种快捷方法，并对其进行了证明。

简单问题

首先，我先来介绍一些比较简单的情况。设

F(s)=s3+s+1(s−1)(s−2)(s−3)F(s)=\frac{s^3+s+1}{(s-1)(s-2)(s-3)}F(s)=(s−1)(s−2)(s−3)s3+s+1​

通过留数法我们可以分解为

F(s)=1+s3+s+1(s−2)(s−3)∣s=1⋅1s−1+s3+s+1(s−1)(s−3)∣s=2⋅1s−2+s3+s+1(s−1)(s−2)∣s=3⋅1s−3=1+32⋅1s−1−11s−2+312⋅1s−3\begin{aligned} F(s) &= 1+\frac{s^3+s+1}{(s-2)(s-3)}\bigg|_{s=1} \cdot \frac{1}{s-1} \\ &\quad +\frac{s^3+s+1}{(s-1)(s-3)}\bigg|_{s=2}\cdot \frac{1}{s-2} \\ &\quad +\frac{s^3+s+1}{(s-1)(s-2)}\bigg|_{s=3}\cdot \frac{1}{s-3} \\ &= 1+\frac{3}{2}\cdot\frac{1}{s-1}-\frac{11}{s-2}+\frac{31}{2}\cdot\frac{1}{s-3} \end{aligned}F(s)​=1+(s−2)(s−3)s3+s+1​​s=1​⋅s−11​+(s−1)(s−3)s3+s+1​​s=2​⋅s−21​+(s−1)(s−2)s3+s+1​​s=3​⋅s−31​=1+23​⋅s−11​−s−211​+231​⋅s−31​​

这种方法虽然直接，但是对于更复杂的情况（特别是分母含有二次式时），这种方法就无法使用了，因此我们需要更加系统的方法。

分母含有二次式的情况

对于分母含有不可约二次式的有理分式，我们可以通过类似的思路进行分解。 设有下列多项式

F(s)=ds2+es+f(s2+as+b)(s−c)F(s) = \frac{ds^2 + es + f}{(s^2 + as + b)(s - c)}F(s)=(s2+as+b)(s−c)ds2+es+f​

则我们可以设：

F(s)=As+Bs2+as+b+Cs−cF(s) = \frac{As + B}{s^2 + as + b} + \frac{C}{s - c}F(s)=s2+as+bAs+B​+s−cC​

快捷求解方法

关键的快捷方法在于：

  1. 求解 CCC：直接使用留数法

    C=lim⁡s→c(s−c)F(s)=ds2+es+fs2+as+b∣s=cC = \lim_{s \to c} (s - c)F(s) = \frac{ds^2 + es + f}{s^2 + as + b}\bigg|_{s=c}C=s→clim​(s−c)F(s)=s2+as+bds2+es+f​​s=c​

  2. 求解 AAA 和 BBB：可以分成三种情况

    情况一 当 c=0c=0c=0 时，可以直接使用通分法求解 AAA 和 BBB：

    F(s)=As2+Bs+C(s2+as+b)s(s2+as+b)F(s) = \frac{As^2 + Bs+C(s^2 + as + b)}{s(s^2 + as + b)}F(s)=s(s2+as+b)As2+Bs+C(s2+as+b)​

    然后比较系数即可。

    情况二 当 c≠0c\neq 0c=0 时，可以先带入 s=0s=0s=0 计算得到

    B=b(Cc−fcb)=Cb−fcB = b(\frac{C}{c}-\frac{f}{cb})=\frac{Cb-f}{c}B=b(cC​−cbf​)=cCb−f​

    然后再选择一个合适的其他值代入即可。

A1s−a+A2(s−a)2+⋯+An(s−a)n\frac{A_1}{s - a} + \frac{A_2}{(s - a)^2} + \cdots + \frac{A_n}{(s - a)^n}s−aA1​​+(s−a)2A2​​+⋯+(s−a)nAn​​

基本原理

有理分式分解基于以下数学原理：

定理：设 F(s)=P(s)Q(s)F(s) = \frac{P(s)}{Q(s)}F(s)=Q(s)P(s)​ 是真分式（deg⁡P<deg⁡Q\deg P < \deg QdegP<degQ），且 Q(s)Q(s)Q(s) 可以因式分解为：

Q(s)=(s−a1)n1(s−a2)n2⋯(s−ak)nk(s2+b1s+c1)m1⋯Q(s) = (s - a_1)^{n_1}(s - a_2)^{n_2}\cdots(s - a_k)^{n_k}(s^2 + b_1s + c_1)^{m_1}\cdotsQ(s)=(s−a1​)n1​(s−a2​)n2​⋯(s−ak​)nk​(s2+b1​s+c1​)m1​⋯

则 F(s)F(s)F(s) 可以唯一地分解为部分分式之和。

留数法的证明

对于简单极点 s=cs = cs=c，我们有：

F(s)=ds2+es+f(s2+as+b)(s−c)=As+Bs2+as+b+Cs−cF(s) = \frac{ds^2 + es + f}{(s^2 + as + b)(s - c)} = \frac{As + B}{s^2 + as + b} + \frac{C}{s - c}F(s)=(s2+as+b)(s−c)ds2+es+f​=s2+as+bAs+B​+s−cC​

两边同时乘以 (s−c)(s-c)(s−c)：

ds2+es+fs2+as+b=(As+B)s−cs2+as+b+C\frac{ds^2 + es + f}{s^2 + as + b} = (As + B)\frac{s-c}{s^2 + as + b} + Cs2+as+bds2+es+f​=(As+B)s2+as+bs−c​+C

令 s→cs \to cs→c，左边第一项趋于 0，得：

C=lim⁡s→cds2+es+fs2+as+b∣s=c=dc2+ec+fc2+ac+bC = \lim_{s \to c} \frac{ds^2 + es + f}{s^2 + as + b}\bigg|_{s=c} = \frac{dc^2 + ec + f}{c^2 + ac + b}C=s→clim​s2+as+bds2+es+f​​s=c​=c2+ac+bdc2+ec+f​

这就是留数法的理论基础。

情况二公式的证明

当 c≠0c \neq 0c=0 时，我们有快捷公式：

B=Cb−fcB = \frac{Cb - f}{c}B=cCb−f​

证明：将分解式通分：

As+Bs2+as+b+Cs−c=(As+B)(s−c)+C(s2+as+b)(s2+as+b)(s−c)\frac{As + B}{s^2 + as + b} + \frac{C}{s - c} = \frac{(As + B)(s - c) + C(s^2 + as + b)}{(s^2 + as + b)(s - c)}s2+as+bAs+B​+s−cC​=(s2+as+b)(s−c)(As+B)(s−c)+C(s2+as+b)​

展开分子：

(As+B)(s−c)+C(s2+as+b)=As2−Acs+Bs−Bc+Cs2+Cas+Cb(As + B)(s - c) + C(s^2 + as + b) = As^2 - Acs + Bs - Bc + Cs^2 + Cas + Cb(As+B)(s−c)+C(s2+as+b)=As2−Acs+Bs−Bc+Cs2+Cas+Cb =(A+C)s2+(−Ac+B+Ca)s+(−Bc+Cb)= (A + C)s^2 + (-Ac + B + Ca)s + (-Bc + Cb)=(A+C)s2+(−Ac+B+Ca)s+(−Bc+Cb)

与原分子 ds2+es+fds^2 + es + fds2+es+f 比较常数项：

−Bc+Cb=f-Bc + Cb = f−Bc+Cb=f Cb−Bc=fCb - Bc = fCb−Bc=f

因为 c≠0c \neq 0c=0，两边同时除以 ccc：

B=Cb−fc=f+Bc−Cbc=fc+B−B=Cb−fcB = \frac{Cb - f}{c} = \frac{f + Bc - Cb}{c} = \frac{f}{c} + B - B = \frac{Cb - f}{c}B=cCb−f​=cf+Bc−Cb​=cf​+B−B=cCb−f​

整理得：

B=Cb−fcB = \frac{Cb - f}{c}B=cCb−f​

这就是情况二公式的来源。代入 s=0s=0s=0 可直接得到此结果。

情况一的说明

当 c=0c = 0c=0 时，上述公式不适用（分母为0），此时需要直接通分比较系数。这就是为什么要分两种情况讨论的原因。

实用技巧总结

一般步骤

对于形如 F(s)=ds2+es+f(s2+as+b)(s−c)F(s) = \frac{ds^2 + es + f}{(s^2 + as + b)(s - c)}F(s)=(s2+as+b)(s−c)ds2+es+f​ 的有理分式：

  1. 第一步：求一次项系数

    使用留数法：

    C=dc2+ec+fc2+ac+bC = \frac{dc^2 + ec + f}{c^2 + ac + b}C=c2+ac+bdc2+ec+f​

  2. 第二步：求二次项系数

    • 当 c=0c = 0c=0 时（情况一）：
      • 通分后比较各项系数
      • 建立方程组求解 AAA 和 BBB
    • 当 c≠0c \neq 0c=0 时（情况二）：
      • 先用公式 B=Cb−fcB = \frac{Cb - f}{c}B=cCb−f​ 快速求出 BBB
      • 再选择一个合适的值（如 s=−1s = -1s=−1 或 s=−as = -as=−a）代入求 AAA

  3. 第三步：验证

    代入特殊值或通分验证结果的正确性

特殊技巧

  1. 分母全为一次因子：直接对每个极点使用留数法即可

  2. 分母含有重根：使用导数公式

    Ak=1(n−k)!lim⁡s→adn−kdsn−k[(s−a)nF(s)]A_k = \frac{1}{(n-k)!} \lim_{s \to a} \frac{d^{n-k}}{ds^{n-k}}\left[(s-a)^n F(s)\right]Ak​=(n−k)!1​s→alim​dsn−kdn−k​[(s−a)nF(s)]

  3. 选择代入值的技巧：

    • 优先选择 s=0s = 0s=0（如情况二中求 BBB）
    • 其次选择 s=−1,−2s = -1, -2s=−1,−2 等简单值
    • 避免选择使分母为 0 的值

因为 c=−1≠0c = -1 \neq 0c=−1=0，使用情况二的公式求 BBB：

F(s)=−1s2+2s+2+2s+1F(s) = \frac{-1}{s^2+2s+2} + \frac{2}{s+1}F(s)=s2+2s+2−1​+s+12​

总结

本文系统介绍了有理分式分解的快捷方法：

  1. 留数法：适用于求所有一次项系数，计算简便
  2. 情况一（c=0c=0c=0）：通分比较系数法，适用于分母含有 sss 的情况
  3. 情况二（c≠0c\neq 0c=0）：利用快捷公式 B=Cb−fcB = \frac{Cb - f}{c}B=cCb−f​，大大简化计算
  4. 导数法：处理重根情况
  5. 所有方法都有严格的数学证明支持

这些方法在信号与系统、自动控制、电路分析等工程学科中都有广泛应用，特别是情况二的快捷公式可以显著提高计算效率。熟练掌握这些技巧，可以在考试和实际工程计算中节省大量时间。
